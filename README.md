# Travail Personnel de l'Etudiant : Oumaté Alim Alhadji

Ce repository est le resultat d'un TPE donnée à l'ENSPM - Cameroun dans le cadre de l'UE : Algorithmique et Programmation en C/Java. 

## Principe

Ce projet est une collection de fonctions mathématiques de base écrites en C. Il inclut un programme principal `main.c` qui permet de tester ces fonctions via un menu interactif. Les fonctions sont implémentés et déclarés dans les fichiers `bibliotheque.c` et `bibliotheque.h`.

## Fonctions Implémentées

1.  `fact` : Calcule la factorielle d'un nombre entier.
2.  `power` : Calcule la puissance d'un nombre entier.
3.  `permutation` : Permute les valeurs de deux entiers.
4.  `isPrimeNumber` : Vérifie si un nombre entier positif est premier.
5.  `solverEquation` : Résout une équation du second degré ($ax^2 + bx + c = 0$).
6.  `distancePoint` : Calcule la distance entre deux points en utilisant une structure.

## Compillation et exécution

Executer cette commande pour compiler.
```sh
gcc main.c bibliotheque.c -o program
```

Executer cette commande pour lancer le programme.
```sh
./program
```
